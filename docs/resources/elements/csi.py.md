# resources/elements/csi.py

## Propósito

Define elementos e seletores utilizados para automação no sistema Projudi.

Este módulo contém variáveis com URLs, seletores CSS e XPath para facilitar
operações automatizadas de login, busca de processos, navegação e manipulação
de documentos no Projudi.

## Constantes

- `URL_LOGIN`
- `URL_BUSCA_CHAMADO`
- `URL_CONFIRMA_LOGIN`
- `XPATH_CAMPO_USERNAME`
- `XPATH_CAMPO_SENHA`
- `XPATH_BTN_ENTRAR`
- `XPATH_INPUT_NUMERO_CHAMADO`
- `XPATH_BTN_BUSCAR`
- `XPATH_TABLE_SOLICITACOES`
- `XPATH_DIV_POPUP_ANEXOS`
- `XPATH_IFRAME_ANEXOS`
- `COMMAND_ANEXOS`
